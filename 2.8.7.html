2.8.7 Акции "Купи X - Получи Y Бесплатно"

Схемы предложений типа "купи один - получи один бесплатно", где пользователь платит только за самый дорогой товар, могут быть использованы для получения неприменимых товаров бесплатно или оплаты самых дешевых товаров с получением более дорогих бесплатно.

Возможные логические уязвимости:

  • Не применяется скидка на самый дешевый товар в акции "3 по цене 2"
  • Акция "3 по цене 2" может превратиться в "2 по цене 1"
  • Акция "3 по цене 2" может стать скидкой 33% на всю корзину
  • Акция "3 по цене 2" может превратиться в "4 по цене 2"
  • Акция "3 по цене 2" может стать "3 по цене 1" из-за ошибки в ПО
